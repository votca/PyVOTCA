
############
votcapytools
############

Generic tools to interact with Votca using Python


Installation
------------

To install votcapytools, do:

.. code-block:: console

  git clone https://github.com/votca/PyVOTCA.git
  cd pyvotca
  pip install .


Run tests (including coverage) with:

.. code-block:: console

  python setup.py test
